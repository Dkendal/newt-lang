a unittest can reference a top-level type alias by name

=======

type Foo as 1

unittest "alias" do
  assert Foo <: number
  assert Foo == 1
  assert not (Foo <: string)
end

=======

type Foo = 1;

=======
unittest "alias"
  ok      Foo <: number
  ok      Foo == 1
  ok      not (Foo <: string)

3 assertion(s): 3 passed, 0 failed
